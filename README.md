## Run Locally

Clone the project

```bash
  git clone https://github.com/razzsnwr8/webapck_setup.git
```

Go to the project directory

```bash
  cd webapck_setup
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
